## Part A — Download + install Zotero 6 (10 min)

```bash
mkdir -p /tmp/zotero-install && cd /tmp/zotero-install && \
  wget -q "https://www.zotero.org/download/client/dl?channel=release&platform=linux-x86_64" \
  -O zotero.tar.bz2 && \
  tar -xjf zotero.tar.bz2 && \
  ls zotero*/ | head -20
```

Expected: a `zotero-linux-x86_64/` folder appears. To run it:

```bash
cd zotero-linux-x86_64 && ./zotero &
```

If a desktop launcher is wanted (auto-start Zotero):

```bash
./zotero --install-app
```

This places a `Zotero.desktop` file in `~/.local/share/applications/` so it shows up
in your application menu.

## Part C — Install Better BibTeX (CRITICAL — 5 min)

Better BibTeX gives us the `auth + year + title` citation keys LaTeX expects.
Without it, Overleaf citations break randomly.

1. In Zotero: **Tools → Add-ons** (or **`Ctrl+Shift+A`** shortcut)
2. Top-right **gear icon ⚙** → **Install Add-on From File…**
3. Download the plugin (still in browser):
   `https://retorque.re/zotero-better-bibtex/installation/`
   - Click the latest green "Better BibTe.x.x.x.xpi" link
   - That `.xpi` is the plugin file
4. Back in the Add-ons dialog, select the downloaded `.xpi` file → **Install**
5. Zotero prompts **"Restart Zotero"** → click **Restart Now**
6. After restart: **Edit → Settings → Better BibTeX → Export**
   - Set **Citation key format** to: `auth + year + title`
   - Click **Apply**

Verification: in your empty library, do **`Shift+Ctrl+C`** or **right-click → Generate BibTeX Citation Key**.
A key like `smith2024shap` should appear (we'll see real examples in Phase 2).

## Part D — Install Zotero Connector browser extension (3 min)

The Connector is a small icon that appears in your browser address bar.
On Google Scholar, PubMed, arXiv, IEEE Xplore, etc., clicking it saves the paper
to your library with one click.

1. Open Firefox
2. Visit Firefox Add-ons → search "Zotero Connector"
   - Or direct link: `https://addons.mozilla.org/en-US/firefox/addon/zotero-connector/`
3. Click **Add to Firefox** → **Add**
4. A new icon (looks like a red Z) appears in the address bar. Pin it (right-click → Pin to Toolbar).
5. Click the icon once in any tab — it might ask you to confirm Save Location. Pick
   **`Thesis - XAI Ensemble Comparison`** as the default.

## 🆘 Common Zotero-on-Linux pitfalls

| Symptom | Fix |
|---|---|
| `./zotero: command not found` after extract | You're not inside the `zotero-linux-x86_64/` directory; `cd` into it first |
| Zotero opens but immediately crashes | Install missing libs: `sudo apt install libnss3 libgbm1 libasound2` |
| Better BibTeX doesn't appear in Add-ons after install | Restart Zotero *fully* (close window + tray icon), then reopen |
| `references.bib` is empty after export | The library was empty. Make sure Step E actually saved 5 papers |
| Firefox Connector icon stays gray | Open `about:preferences#general`, allow site permissions for `scholar.google.com` |
